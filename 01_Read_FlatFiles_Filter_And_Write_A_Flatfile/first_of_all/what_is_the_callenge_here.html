Grobe Beschreibung

In diesem Spring Batch Prozess werden eine oder mehrere Eingabesätze über mehrere CSV Eingabedateien eingelesen und in einer CSV-Gesamtdatei abgespeichert.

Programmablauf

Input (ItemReader)

Lesen einer oder mehrere Eingabesätze über mehrere CSV Eingabedateien
Die Verarbeitung erfolgt stückweise (‘Chunk-oriented‘ - hier 5 Datensätze

Process (ItemProcessor)

Eingabesätze werden gefiltert (Nur die erste Kopfzeile wird übernommen)
Der Prozessor übergibt eine Liste von Personenitems an den Writer (stückweise)

Output (ItemWriter)

Es wird eine neue CSV Ausgabedatei (mit Zeitstempel im Dateinamen) erzeugt mit allen Eingabesätzen.
